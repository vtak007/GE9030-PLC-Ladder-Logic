# GE9030 PLC Ladder Logic

GE 9030 PLC ladder logic program, schematics, and controls analysis documentation for the PP-148 unit (PipePro III).

---

|---|---|
| `PP-148.xml` | PLC program export (GE 9030 / PAC Machine Edition format) |

| `PP-148 PCU Program.zip` | Archived PCU program for the PP-148 unit |
| `PP-148 Controls Analysis Report.pdf` | Controls analysis report (final/reference) |
| `PP-148 Controls Analysis Report.docx` | Controls analysis report (editable) |
| `Hawkeye schematic.png` | Electrical schematic image |

## Notes

The editable program files are `PP-148.xml` and `PP-148.SwxCF` — all PDFs and PNGs are read-only reference material. The `.xml` export can be opened in GE's PAC Machine Edition software.
